Texas Historical Markers

  • Click here to see it running on Heroku
  • 12k+ Historical Markers searchable and displayed inside google maps
  • Search markers by location using HTML5 Geolocation or Google reverse geocoding
  • Search markers text, powered by pg_search and run against title, tags, and main text
  • Secured API endpoint at /api/geo, given lat, lng and optional radius returns json or xml marker data
  • API secured using access tokens
  • Data provided by the State of Texas, which I converted from UTM to GPS
  • Ruby on Rails and PostgreSQL
