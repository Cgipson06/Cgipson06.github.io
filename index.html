Texas Historical Marker

This app started as a simple marker viewer, and has expanded over time to include geolocation/reverse geolocation searches, full text search, and the ability to flag, correct, and log errors present in the original state database.

App Overview
  • 12k+ Historical Markers searchable and displayed inside google maps
  • Location based searches utilize HTML5 Geolocation or Google Reverse Geocoding API
  • Fast Full Text search of markers powered by PostgreSQL, utilizes tsvector column for faster queries
  • Users can create accounts to save markers, as well as submit error reports
  • Users can login or create accounts via Facebook
  • Admin interface for viewing error reports and editing marker data
  • Secured API endpoint at /api/geo, given lat, lng and optional radius returns json or xml marker data
  • Ruby on Rails,PostgreSQL,SCSS,jQuery
Running on Heroku at TxHistoricalMarker.com Running at TxHistoricalMarker.com

ATXGreenbelt

This site was inspired by the lack of a quick and easy to use tool to check the water conditions in the Barton Creek Greenbelt.

Site Overview
  • Utilizes USGS water API for current condition @ Barton Creek Greenbelt
  • Medium and Large screen navigation via WheelNav.js
  • Charting of USGS data via Highcharts.js
  • HTML/CSS + jQuery + Bootstrap
